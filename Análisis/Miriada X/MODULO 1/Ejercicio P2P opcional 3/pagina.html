Comentario:

Hay que calcular la tupla (A,B,C,D) garadora de todas las reglas CSS que compiten. A tiene maximo peso y D minimo. Si hay empate en A, se mira en B y asi sucesivamente.

  • A = estilo en línea
  • B = número de IDs (en nuestro caso = caja)
  • C = número de clases (en nuestro caso = cabecera)
  • D = número de marcas HTML (en nuestro caso = header y h1)

    • PUNTUACIONES

  • h1 {color: purple;}
  • h1 {color: grey;}

    • Puntuacion A=0 B=0 C=0 D= 1 TOTAL: 0,0,0,1

  • header h1 {color: lime; }

    • Puntuacion: A=0 B=0 C=0 D=2 TOTAL: 0,0,0,2

  • #caja .cabecera h1 {color:red ; }

    • Puntuacion: A=0 B=1 C=1 D=1 TOTAL: 0,1,1,1 ¡GANADORA!

  • #caja header h1 {color:blue;}

    • Puntuacion: A=0 B=1 C=0 D=2 TOTAL: 0,1,0,2
